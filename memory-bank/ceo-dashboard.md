## Dashboard Maintenance Guidelines

**Update Frequency:** After each project status change or at minimum daily

**Update Triggers:**
- Project status changes (kickoff, milestone completion, blocker added/removed)
- New action items identified
- Ownership transfers
- Blocker resolution
- Milestone achievements

**Sync with activeContext.md (MANDATORY - Dual-Update Rule):**
This dashboard provides the at-a-glance operational view ("what's next"). For full strategic context, historical background, and detailed status updates ("why and how"), always update the corresponding section in activeContext.md simultaneously.
